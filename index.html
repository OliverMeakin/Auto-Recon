What it does

Faster matching, clearer outputs, fewer surprises during close.

Finds matches

Pairs and subset-matches transactions that net to a target amount within a grouping you define.

Applies guardrails

Enforce constraints like allowed sources, type separation, and one-to-many vs many-to-one behavior.

Creates traceability

Writes statuses back, appends cleared groups to a log sheet, and generates a debug trace for review.

Note: This is positioned as a configurable engine. No company-specific labels are required.

How it works

A review-first flow that fits existing processes and improves audit clarity.

1

Import your export

Use the same Excel export you already reconcile. Map the relevant columns once.

2

Run matching rules

Exact matches first, then controlled subsets. Tolerance edge cases are flagged, not forced.

3

Review + clear

Clear groups are appended to a log. Exceptions stay in place with a status for follow-up.
